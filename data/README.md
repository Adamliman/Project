
# House Prices Dataset

File: house_data.xlsx

Description:
This dataset contains information about houses sold, with approximately 21,613 rows and the following columns:

- id: unique identifier
- date: sale date
- price: sale price
- bedrooms: number of bedrooms
- bathrooms: number of bathrooms
- sqft_living: living area in square feet
- sqft_lot: lot size in square feet
- floors: number of floors
- waterfront: presence of waterfront (0/1)
- view: view rating (0-4)
- condition: overall condition of the house (1-5)
- grade: construction quality (1-13)
- sqft_above: square footage above ground
- sqft_basement: square footage of basement
- yr_built: year built
- yr_renovated: year renovated
- zipcode: postal code
- lat: latitude
- long: longitude
- sqft_living15: average living area of 15 nearest neighbors
- sqft_lot15: average lot size of 15 nearest neighbors
- is_renovated: 1 if renovated, 0 otherwise
- year_sale: year of sale
- month_sale: month of sale
- age: age of the house at the time of sale
